A software rasteriser in Nim and demo project (main.nim).
Note: threading.nim is not used and is merely a reference implementation of multithreading created when devising the threading in renderer.nim

Depends on SRWWL/SRWinWL (available at: https://github.com/othd06/SRWWL/tree/main and https://github.com/othd06/SRWinWL/tree/main respectively)
the required files are already included but check out those projects if you're interested.
